To implement this feature follow the steps bellow.
Import Dithering Effect folder directly in the Assets folder.
Right click in the Inspector> add Volumetric Sphere (global) > Add Component. Pick Custom> Sphere Volume Component (Make sure intensity = 1)
Go to Settings in Assets Folder > PC_Renderer> Add Renederer Feature> Dithering Render Feature.
In material, add Dither Shader Mat which was made from the "Dither Shader".
Enjoy :)
